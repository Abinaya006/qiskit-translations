Menginstal Qiskit¶

Persyaratan¶

Qiskit mendukung Python 3.5 atau setelahnya. Namun, baik Python maupun Qiskit adalah ekosistem yang terus berubah, dan terkadang ketika ada perubahan pada salah satunya, ada kemungkinan terjadi masalah kompatibilitas.

Jika Anda mengalami kesulitan menginstal atau menggunakan Qiskit setelah memperbarui versi Python, periksa metadata paket Qiskit untuk Programming Language untuk melihat versi spesifik Python yang didukung.

Kami merekomendasikan menginstal Anaconda, sebuah distribusi Python cross-platform untuk komputasi ilmiah. Jupyter, termasuk di dalam Anaconda, direkomendasikan untuk berinteraksi dengan Qiskit.

Qiskit sudah diuji dan didukung pada sistem 64-bit berikut:

  • Ubuntu 16.04 atau setelahnya

  • macOS 10.12.6 atau setelahnya

  • Windows 7 atau setelahnya

Instalasi¶

Kami merekomendasikan penggunaan lingkungan virtual Python untuk memisahkan Qiskit secara rapi dari aplikasi lain dan meningkatkan pengalaman penggunaan Anda.

Cara paling sederhana untuk menggunakan lingkungan/environment adalah dengan menggunakan perintah conda, yang sudah tersedia dalam Anaconda. Sebuah lingkungan Conda memungkinkan Anda untuk menentukan versi spesifik dari Python dan kumpulan pustaka/library yang ingin digunakan. Buka jendela terminal pada direktori di mana Anda ingin bekerja.

Sebaiknya Anda menggunakan Anaconda prompt yang terinstal bersama dengan Anaconda. Yang harus Anda lakukan hanyalah membuat sebuah lingkungan virtual di dalam Anaconda dan mengaktifkan lingkungan tersebut. Perintah-perintah ini dapat dijalankan di Anaconda prompt terlepas dari mesin Windows atau Linux.

Buat sebuah lingkungan minimal dengan hanya Python yang diinstal di dalamnya.

conda create -n name_of_my_env python=3

Aktifkan lingkungan baru Anda.

conda activate name_of_my_env

Catatan

Jika Anda menggunakan versi conda sebelum 4.6, gunakan: Untuk Linux atau macOS: source activate name_of_my_env Untuk Windows: activate name_of_my_env

Selanjutnya, instal paket Qiskit, yang berisi Terra, Aer, Ignis, dan Aqua.

pip install qiskit

Catatan

Dimulai sejak Qiskit 0.13.0, pip 19 atau yang lebih baru dibutuhkan untuk menginstal qiskit-aer dari precompiled binary di Linux. Jika Anda tidak memiliki pip 19 terinstal Anda dapat menjalankan pip install -U pip untuk memperbaruinya. Tanpa pip 19 atau yang lebih baru perintah ini akan mencoba untuk menginstal qiskit-aer dari sdist (distribusi sumber) yang akan mencoba untuk mengkompilasi aer secara lokal diam-diam.

If the packages were installed correctly, you can run conda list to see the active packages in your virtual environment.

Jika Anda berniat untuk menggunakan fungsionalitas visualisasi atau Jupyter notebook dianjurkan untuk menginstal qiskit dengan persyaratan visualisasi tambahan

pip install qiskit[visualization]

It is worth pointing out that if you're a zsh user (which is the default shell on newer versions of macOS), you'll need to put qiskit[visualization] in quotes:

pip install 'qiskit[visualization]'

Catatan

Setelah Anda telah menginstal dan memverifikasi paket Qiskit yang Anda ingin gunakan, impor mereka ke lingkungan Anda dengan Python untuk mulai bekerja.

import qiskit

Catatan

Jika Anda ingin berkontribusi pada komunitas Qiskit dengan mengembangkan dan mengkontribusikan kode dengan kode Qiskit yang terbaru, lihat Bangun paket Qiskit dari sumber.

Akses Sistem Kuantum IBM¶

IBM Quantum menawarkan beberapa komputer kuantum sungguhan dan simulator komputasi klasik berkinerja tinggi melalui IBM Quantum Experience dengan Qiskit. Ikuti langkah-langkah ini untuk mengatur lingkungan Qiskit Anda untuk mengirim pekerjaan ke sistem IBM Quantum.

Catatan

Sejak rilisnya Qiskit 0.11, jika Anda sebelumnya telah menyimpan kredensial IBM Quantum Experience Anda secara lokal, Anda mungkin perlu memperbarui mereka sehingga Anda dapat menggunakan IBM Quantum Experience V2 yang baru. Lihat Memperbarui Kredensial IBM Quantum Experience Anda.

Untuk mengkonfigurasi akun Anda, Anda akan membuat sebuah file konfigurasi lokal yang menyimpan sebuah kunci API

1. Buat akun gratis IBM Quantum Experience.

2. Navigasi ke My Account untuk melihat pengaturan akun Anda.

3. Klik pada Copy token untuk menyalin token ke papan klip Anda. Tempelkan sementara token API ini ke dalam teks editor favorit Anda sehingga Anda dapat menggunakannya nanti untuk membuat file konfigurasi akun.

4. Jalankan perintah berikut untuk menyimpan token API Anda secara lokal untuk digunakan kemudian hari dalam file konfigurasi yang disebut qiskitrc. Ganti MY_API_TOKEN dengan token API yang Anda simpan dalam teks editor Anda.

from qiskit import IBMQ
IBMQ.save_account('MY_API_TOKEN')

Untuk detil lebih, seperti cara mengelola beberapa kredensial akun IBM Quantum, lihat tutorial ini yang berjudul The IBM Quantum Account.

Memeriksa Versi Mana yang Terinstal¶

Karena paket Qiskit mencakup suatu konstelasi dengan elemen yang berbeda, hanya mencetak versi dengan menjalankan qiskit.__version__ dapat menyesatkan karena itu hanya akan memberikan versi untuk paket qiskit-terra. Hal ini dikarenakan namespace qiskit dalam Python tidak berasal dari paket Qiskit, melainkan merupakan bagian dari paket qiskit-terra.

import qiskit
qiskit.__version__

'0.16.0'

Untuk melihat versi semua elemen Qiskit di lingkungan Anda, Anda dapat gunakan atribut __qiskit_version__. Sebagai contoh, menjalankan perintah berikut akan mengembalikan kamus yang mencakup versi untuk setiap paket Qiskit yang telah diinstal.

qiskit.__qiskit_version__

{'qiskit-terra': '0.16.0',
 'qiskit-aer': '0.7.0',
 'qiskit-ignis': '0.5.0',
 'qiskit-ibmq-provider': '0.11.0',
 'qiskit-aqua': '0.8.0',
 'qiskit': '0.23.0'}

Tip

Jika Anda mengajukan sebuah isu atau perlu membagikan versi Qiskit yang Anda instal untuk sesuatu, gunakan atribut __qiskit_version__.
