DAGDepNode¶

class DAGDepNode(type=None, op=None, name=None, qargs=None, cargs=None, condition=None, successors=None, predecessors=None, reachable=None, matchedwith=None, successorstovisit=None, isblocked=None, qindices=None, cindices=None, nid=- 1)[código fonte]¶

Object to represent the information at a node in the DAGDependency().

It is used as the return value from *_nodes() functions and can be supplied to functions that take a node.
